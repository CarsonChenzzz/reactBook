## 支持

- 豆瓣
- 百度
- 阿里云

## 原理说明

- 运用 React 和 BootStrap 实现前端页面构建


- 使用 Webpack 实现模块加载和打包功能


- 运用 React 中路由功能实现参数传递及页面跳转
- 运用豆瓣 API 加载资源
- 运用阿里云 MySQL 数据库实现评论功能的实现
- 运用百度地图 API 实现地图定位功能

## 下载安装

>npm install

> npm start

> webpack —watch
